# To run to code

Initialize the database
```
  make initdb
```

Setup environment variables
```
  export APP_MYSQL_HOST=${your_mysql_host}
  export APP_MYSQL_USER=${your_mysql_user}
  export APP_MYSQL_PASS=${your_mysql_password}
  export APP_MYSQL_DB=tapclicks
```

Run the application
```
  php index.php
```
